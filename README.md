Un sistema de Punto de Venta (POS) e Inventario de grado profesional, diseñado para la eficiencia y velocidad en entornos de retail y supermercados.

## Características Principales
- **Interfaz Profesional (White/Blue Theme)**: Diseño limpio y de alto contraste enfocado en la usabilidad durante largas jornadas, minimizando la fatiga visual.
- **Teclado Numérico y Acciones en Pantalla**: Soporte completo para pantallas táctiles y teclado virtual integrado.
- **Caja Rápida (POS)**: Lógica optimizada para compatibilidad nativa con escáneres de código de barras láser USB (emulación de teclado directo).
- **Feedback Auditivo Real**: Sonidos de caja registradora clásicos y confirmación de escaneo mediante el sistema de audio nativo de Windows.
- **Dashboard Predictivo**: Analiza el historial de ventas para alertar sobre stock crítico.
- **Gestión de Productos**: Añade y organiza tus productos por categoría fácilmente.

## Tecnologías
- C++ 17
- Qt 6 (QML / Qt Quick)
- CMake
- SQLite (vía abstracción C++)
- Windows Multimedia API (mmsystem)
