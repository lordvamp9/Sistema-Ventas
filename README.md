# Multicosas POS

Un sistema de Punto de Venta (POS) e Inventario moderno, con estética *Glassmorphism* y animaciones fluidas, diseñado para pequeños negocios.

## Características Principales
- **Dashboard Predictivo**: Analiza el historial de ventas para predecir cuándo te quedarás sin stock.
- **Caja Rápida (POS)**: Interfaz optimizada para escáneres de código de barras láser (emulación de teclado).
- **Gestión de Productos**: Añade y organiza tus productos por categoría fácilmente.
- **Diseño Premium**: Interfaz limpia, oscura y translúcida (Glassmorphism), sin elementos visuales distractores.
- **Efectos de Sonido**: Feedback auditivo integrado para lectura de códigos y confirmaciones de pago.

## Tecnologías
- C++ 17
- Qt 6 (QML / Qt Quick)
- CMake
- SQLite (vía abstracción C++)
